# electron-cursor-status

- app 端提供二维码扫描，供手机端访问
- 可选声音提示，app 端设定可记忆，手机端需要手动开启，因为移动端限制必须用户触发才能播放声音
- 手机端通过长连接到 electron ，接受 ws 消息，如果长连接断开，则界面上展示长连接断开状态，并实现重连机制和心跳机制，心跳间隔 10 秒
- 工作中卡片和结束卡片
- 状态历史记录
- 耗时的展示
